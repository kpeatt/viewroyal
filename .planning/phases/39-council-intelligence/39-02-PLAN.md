    **Migration (39-key-votes-table.sql):**
    Create `key_votes` table with schema from RESEARCH.md. Add RLS policy for public read. Add `update_timestamp` trigger. Also ALTER TABLE councillor_highlights ADD COLUMN IF NOT EXISTS `narrative` text (for the enhanced 2-3 paragraph profile). Add `narrative_generated_at` timestamptz column.

    **Key vote detector (key_vote_detector.py):**
    Create `detect_all_key_votes(supabase, person_id=None)` following stance_generator.py patterns.

    Algorithm for each person:
    1. **Fetch all votes** for the person with motion details (result, meeting_date, agenda_item_title)
    2. **Compute vote counts from votes table** (not motions.yes_votes/no_votes -- per Pitfall 5 in RESEARCH.md): `SELECT motion_id, vote, COUNT(*) FROM votes GROUP BY motion_id, vote`
    3. **Minority position detection:** Person voted Yes but motion DEFEATED, or voted No but motion CARRIED/CARRIED AS AMENDED. Use `normalizeMotionResult` logic (upper-case compare).
    4. **Close vote detection:** Total margin (abs(yes - no)) <= 2. Compute from votes table counts.
    5. **Ally break detection:**
       - Compute pairwise alignment for all councillor pairs: % of shared motions where they voted the same way
       - Threshold: pair must normally agree >= 80% of the time
       - Flag when this vote is a disagreement between a usually-aligned pair
    6. **Composite score:** `is_minority * 3 + closeness_factor * 2 + ally_break_count * 1`
       - `closeness_factor`: 1.0 for margin=0, 0.75 for margin=1, 0.5 for margin=2
    7. **Context summary:** Generate with Gemini -- brief one-sentence explanation of why this vote is notable (e.g., "Voted against the 5-2 majority to approve the Helmcken Road rezoning")
    8. **Upsert** into key_votes table with ON CONFLICT (person_id, motion_id) DO UPDATE

    **Tests (test_key_vote_detector.py):**
    - Test minority position detection with mock vote data
    - Test close vote detection (margin 0, 1, 2 are close; 3+ is not)
    - Test ally break detection with mock alignment baselines
    - Test composite score calculation
    - Test that detection works when motions.yes_votes/no_votes are 0 (falls back to votes table counts)
    - Mock Supabase and Gemini calls

    Create `tests/profiling/__init__.py` if it doesn't exist.
